# Контролно 2 – Пример

## I част – Затворени въпроси (по 3 т.)

**1. Какъв ще бъде резултатът?**

```bash
echo "Hello" > file.txt
ln file.txt hard.txt
rm file.txt
cat hard.txt
```

а) Грешка
б) Hello
в) Празен файл
г) hard.txt се изтрива

---

**2. Какъв ще бъде резултатът?**

```bash
echo "Test" > a.txt
ln -s a.txt b.txt
rm a.txt
cat b.txt
```

а) Test
б) Грешка / broken link
в) Празен файл
г) b.txt се изтрива

---

**3. Какъв ще бъде резултатът?**

```bash
VAR=6
echo $((VAR * 3 + 2))
```

а) 20
б) 18
в) 632
г) Грешка

---

**4. Какво прави командата?**

```bash
journalctl -u nginx -f
```

а) Изтрива логовете на nginx
б) Показва логовете на nginx в реално време
в) Показва само грешките на nginx
г) Спира nginx услугата

---

**5. Какво означава буквата `b` в изхода на `ls -l /dev`?**

а) Backup device
б) Block device
в) Binary device
г) Boolean device

---

**6. Какъв ще бъде резултатът?**

```bash
for i in {1..3}; do
    echo $((i * 2))
done
```

а) Извежда: 1 2 3
б) Извежда: 2 4 6
в) Извежда: 1 3 5
г) Грешка

---

**7. Какво проверява операторът `-d` в bash условие?**

а) Файлът е празен
б) Директорията съществува
в) Файлът е изпълним
г) Файлът е четим

---

**8. Какъв ще бъде резултатът?**

```bash
#!/bin/bash
count=1
while [ $count -le 3 ]; do
    echo "Line $count"
    ((count++))
done
```

а) Line 1
б) Line 1, Line 2, Line 3
в) Line 0, Line 1, Line 2
г) Безкраен цикъл

**9. Коя команда показва използваното дисково пространство в четим формат?**

а) `lsblk -h`
б) `df -h`
в) `mount -h`
г) `fdisk -h`

**10. Какво прави следният cron израз?**

б) Изпълнява скрипта всеки понеделник в 8:30
в) Изпълнява скрипта всеки час на 30-та минута
г) Изпълнява скрипта на 8-ми на всеки месец в 30 минути
